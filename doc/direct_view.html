Direct view

 

  Direct calling 'view' method of the class

    Package page::page_conposer provides functioanlity wich allows you to call 'view' method of any other package. You can use it in your ajax scripts to get parts of the page dinamically. To do it send request to the page ./direct_view.html with to GET parameters 'package_name' and 'package_version' with the requesting package's name and version respectively (if version is not specified then it will be defaulted to 'last').
