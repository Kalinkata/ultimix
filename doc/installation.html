Installation

 

  Steps of installation

  First of all you should download distributive of the Ultimix. It can be found here. Unzip archive in any directory on your web-server and run <your_domain>/<unzip_directory>/install.php script. Unzipping it in the root directory may be the easiest way to install Ultimix. After this you will see first screen of the installation wizard.

  Click 'Next' and you'll see the second screen.

  Input database connection settings and click 'Next'.

  In this window you can set main entry point to your site. Mostly allways it can be set unchanged. Click 'Next' once again (the last one)) ).

  After the installation is complete don't forget to delete install.php file and install folder from your server.
