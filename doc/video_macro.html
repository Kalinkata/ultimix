Pre-/post-processing video macro

 

  Postprocessing macro

  Macro: {youtube:<code>}
  Requires: gui package

  This will be replaced with youtube player.

Example:
{youtube:R7yJ-hruqcM}

  Macro: {rutube:<code>}
  Requires: gui package

  This will be replaced with rutube player.

Example:
{rutube:a7999ad76e56e10bbcdbc7abd6c54778}
