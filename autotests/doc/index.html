  RaTM 1.2.0

  A new release of RaTM is available:
    1. Added execution_counter in the test's manifest syntax (check the manual for more information http://ratm.sourceforge.net/manual.html).
    2. According to execution_counter parameter appearance, testing report was extended. Now total test's execution time is outputted and the average time per test iteration.
    3. Command line parameter -test_id was added.
    4. Manifest sintax was extended - added test_id tag (see manual for more information http://ratm.sourceforge.net/manual.html).
    5. Protocol for running tests via HTTP was changed (see ./misc/http_test.php example for more information).
    6. Added new command line parameter -project (see manual for more information).
    7. Row sorting bug in testing report was fixed.
    8. Fiexed bug with external log processing.
    9. Minor changes and improvements.
    10. Refactoring.
